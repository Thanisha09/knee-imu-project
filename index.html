Wearable Dual-IMU System

Real-Time Knee Joint Angle Estimation using IMU & ESP32

🚀 About Project

A wearable system using dual IMU sensors to accurately estimate knee joint angles in real-time. Designed as a low-cost alternative for clinical and rehabilitation monitoring.

⚠️ Problem Statement

📏 Goniometers → Static & user-dependent
📷 Vision systems → Occlusion & lighting issues
🎯 Need → Real-time wearable solution

⚙️ Methodology

Two IMU sensors are placed on the thigh and shank. Data is processed using ESP32 and filtered using a complementary filter to compute stable knee angles in real time.

📊 Results

High accuracy

Low RMSE

Stable readings

High Correlation

Matches reference systems

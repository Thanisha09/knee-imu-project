Wearable Dual-IMU System

Knee Joint Angle Estimation using IMU & ESP32

About Project

This project presents a wearable dual-IMU system for accurate real-time knee joint angle estimation. It provides a low-cost alternative to traditional clinical and vision-based systems.

Problem Statement

Manual Methods

Static, user-dependent, limited accuracy

Vision Systems

Occlusion issues, lighting dependency

Need

Portable, real-time, wearable system

Methodology

Two IMU sensors are placed on the thigh and shank. Data is processed using ESP32 and filtered using a complementary filter to calculate accurate knee angles.

Results

MAE

Low error across all angles

RMSE

Consistent accuracy

Correlation

Strong agreement with reference methods

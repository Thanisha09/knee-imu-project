Wearable Dual-IMU System

Knee Joint Angle Estimation Project

About Project

This project develops a wearable system using IMU sensors and ESP32 to measure knee angles in real-time.

Problem Statement

Traditional methods like goniometers are static, and vision systems suffer from occlusion. A wearable solution is needed.

Methodology

Two IMU sensors are placed on the thigh and shank. Data is processed using ESP32 and angle is calculated using sensor fusion.

Results

System shows low MAE and strong correlation with standard methods.

Wearable Dual-IMU System

Real-Time Knee Joint Angle Estimation using IMU & ESP32

🚀 About Project

A wearable dual-IMU system for accurate real-time knee angle estimation, designed as a low-cost alternative for clinical monitoring.

🧠 System Setup

IMU sensors attached to thigh and shank.

⚠️ Problem Statement

📏 Goniometer → Static & user dependent
📷 Vision → Occlusion issues
🎯 Need → Real-time wearable system

⚙️ Methodology

Two IMU sensors are placed on the thigh and shank. Data is processed using ESP32 and filtered using a complementary filter to compute stable knee angles.

📊 Results

IMU vs reference comparison showing strong correlation.

📈 Performance

Low MAE

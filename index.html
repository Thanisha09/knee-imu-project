Wearable Dual-IMU System

Knee Joint Angle Estimation using IMU & ESP32

🚀 About Project

A wearable system using dual IMU sensors to estimate knee joint angles in real-time for rehabilitation and monitoring.

🧠 System Setup

IMU sensors placed on thigh and shank for angle estimation.

⚠️ Problem Statement

📏 Goniometer → Static & user dependent
📷 Vision → Occlusion issues
🎯 Need → Wearable real-time system

⚙️ Methodology

Two IMU sensors are attached to the leg. Data is processed using ESP32 and filtered to calculate knee angle accurately.

📊 Results & Graph

Comparison of IMU vs reference showing strong correlation.

📈 Performance

Low MAE
Low RMSE
High Accuracy
